# Projeto Base – Angular + Tailwind
Este projeto serve como base para a construção da sua aplicação. Utilize-o como ponto de partida para organizar components, services e toda a estrutura do sistema.

## Como criar um Service:

* Use o comando: ng g s nome-do-service

## Como criar um Component:

* Use o comando: ng g c nome-do-component

## Como rodar a aplicação:
* Use o comando: ng s -o
(O parâmetro -o faz o navegador abrir automaticamente.)
